## Methodik in Kürze

Ausgewertet wurden **ausschliesslich öffentlich zugängliche Quellen** (Bewertungsportale, Kundenzufriedenheits-Umfragen, Branchenstudien). Zwei Wahrnehmungs-Ebenen werden bewusst getrennt: repräsentative Befragungen (faires Gesamtbild) und unaufgeforderte Bewertungskanäle (selbstselektiv, zeigen aber, wo es klemmt). Alle Kennzahlen sind mit Quelle und Konfidenz hinterlegt.
